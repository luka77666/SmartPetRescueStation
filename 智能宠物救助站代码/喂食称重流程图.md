## 流程说明

### 1. 启动方式
- **APP远程控制**: 发送 `MF0` 命令 → `Food_Flag = 1`
- **物理按键**: 手动模式下光标在喂食位，按 KEY_3 → `Food_Flag = 1`
- **定时自动触发**: `AutoControl()` 检测到定时匹配 → `Food_Flag = 1`

### 2. 首次启动初始化
- 记录启动瞬间的初始重量 `start_weight`
- 发送语音播报指令
- 重置防卡粮状态 `anti_jam_state = 0`
- 电机正转5圈(1800度)启动出粮

### 3. 防卡粮自动反转
- 电机正转5圈完成后 → 自动反转1圈(360度)
- 反转1圈完成后 → 自动正转5圈
- 如此循环，无停顿，防止粮仓卡粮

### 4. 增量称重判定（核心停止条件）
- **判定公式**: `当前重量 >= start_weight + target`
- **target 取值**:
  - 定时触发时: `target = schedule[n].feed_weight`（对应定时的独立喂食量）
  - 手动启动时: `target = weight_Value`（全局称重阈值）
- 增量判定的优势: 解决余粮误判问题，只计算本次喂食新增的重量

### 5. 停止方式
- **自动停止**: 增量达到目标值后自动停止
- **手动停止**: APP发送 `MF1` 或按 KEY_4，随时可停止
- 停止时清除所有喂食标志，立即停止电机 `MOTOR_ABORT()`
