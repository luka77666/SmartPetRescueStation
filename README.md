# 基于STM32的智能宠物救助站系统

## 项目简介
本项目设计并实现了一款基于STM32F103C8T6的智能宠物救助站，配套Android APP实现远程监控与智能喂养。系统集成多路传感器、步进电机驱动、WiFi通信和APP控制，具备自动/手动双模式运行能力。

## 功能特性
- 🔥 **实时环境监测**：DHT11温湿度、HX711称重、水位传感器，数据每5秒刷新
- 🍖 **智能喂食控制**：自动/手动双模式，增量称重算法，防卡粮机制
- 💧 **自动补水**：水位低于阈值自动启动水泵
- ⏰ **定时喂食**：3组独立定时任务，可分别设置时间和喂食量
- 📱 **APP远程控制**：Android客户端，TCP Socket通信，实时状态显示
- 🔗 **WiFi联网**：ESP8266 AP模式，手机直连，无需路由器

## 硬件架构
| 模块 | 型号 | 接口 |
|------|------|------|
| 主控 | STM32F103C8T6 | Cortex-M3 @72MHz |
| 温湿度 | DHT11 | 单总线 |
| 称重 | HX711 + 称重传感器 | 24位ADC |
| 水位 | 水位传感器 | ADC（PA1） |
| 电机 | 28BYJ-48 | ULN2003驱动 |
| 水泵 | 5V微型水泵 | 继电器控制 |
| 通信 | ESP8266-01S | UART |
| 显示 | ST7735 TFT 1.8寸 | SPI |

## 软件架构
```
Android APP（Kotlin + Compose）
        ↕ TCP Socket（端口5000）
ESP8266 WiFi（AP模式 / SSID: ZNCWJZZ）
        ↕ UART
STM32F103C8T6（Keil / C语言）
        ↕ GPIO/I2C/SPI/ADC
传感器模块 + 执行机构
```

## 技术要点
- **增量称重算法**：记录喂食前初始重量，通过增量判断停止条件，精度达0.1g
- **防卡粮机制**：电机正转5圈→反转1圈循环，解决饲料卡堵
- **电源完整性**：470μF电容储能+PWM缓启动，解决电机瞬态压降
- **数字滤波**：10点均值滤波+防抖计数器，消除振动对称重干扰

## 快速开始
### 硬件准备
1. 按原理图连接各模块
2. 使用ST-Link烧录固件
3. 接通5V/2A电源

### APP使用
1. 手机连接热点 `ZNCWJZZ`（无密码）
2. 打开APP，输入IP `192.168.4.1`
3. 点击连接，开始监控
